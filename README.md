# ScenTalk - Premium Perfume E-Commerce Website

## Overview
ScenTalk is a modern, responsive e-commerce website for selling premium perfumes. Built with HTML, CSS, and JavaScript, it provides a clean and intuitive shopping experience.

## Features

✨ **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices

🛍️ **Product Catalog** - Browse a curated collection of premium perfumes

🛒 **Shopping Cart** - Add/remove items and view your cart

📝 **Contact Form** - Get in touch with customer support

💳 **Checkout Integration Ready** - Ready to integrate with payment gateways

## Project Structure

```
├── index.html       # Main HTML file
├── styles.css       # CSS styling
├── script.js        # JavaScript functionality
└── README.md        # This file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start shopping!

## Customization

### Adding Products
Edit the `products` array in `script.js` to add more perfumes:

```javascript
const products = [
    { id: 1, name: "Product Name", price: 99.99, description: "Product description", emoji: "🌹" },
    // Add more products here
];
```

### Styling
Modify `styles.css` to customize colors, fonts, and layout.
